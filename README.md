# StudyGenie

AI-powered study planner built with Django.

## Features
- User Authentication
- Personalized Study Plan Generation
- Dashboard with Progress Tracking
- Clean SaaS-style UI

## Tech Stack
- Python
- Django
- SQLite
- HTML/CSS/JavaScript

## Setup

1. Clone repository
2. Create virtual environment
3. Install dependencies:
   pip install -r requirements.txt
4. Run server:
   python manage.py runserver
